此项目基于Yunzai的个人自用插件

## 安装教程 

>Yunzai 根目录执行命令安装

 - Github
``` bash 
git clone --depth=1 https://github.com/Lovely02Y/Lovely-mysMap.git ./plugins/Lovely-mysMap/
```

>安装依赖

``` bash 
pnpm install
```

## 安装教程 

#### Yunzai 根目录执行命令安装

 - Github
``` bash 
git clone --depth=1 https://github.com/Lovely02Y/Lovely-mysMap.git ./plugins/Lovely-mysMap/
```
> [!NOTE]
> 如果你的网络环境较差，无法连接到 Github，可以使用 [GitHub Proxy](https://mirror.ghproxy.com/) 提供的文件代理加速下载服务
>
